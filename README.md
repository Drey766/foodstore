## About the Project

This project simulates a real restaurant website where business owners can:
- Update menu items
- Change prices
- Mark dishes as popular
- Upload images
- Manage content without touching code

The goal is to demonstrate practical frontend skills and real-world problem solving for small businesses.

---

## Planned Features

- Admin authentication
- Admin dashboard for menu management
- Online reservations
- Contact form
- Blog / announcements section
